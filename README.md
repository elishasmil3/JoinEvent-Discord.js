
# Welcome Discord Bot

Simple Discord Bot handling Joining and Leaving Events in a server.



## How to Use -

### 1. Download the Code

### 2. Edit the example.env

    DISCORD_TOKEN=<TOKEN>
    GUILD_ID=
    CLIENT_ID=
    WELCOME_CHANNEL=

### 3. Run the Code
